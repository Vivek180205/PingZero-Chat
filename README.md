# PingZero Chat

A real-time chat application built using (tech stack).

## Tech Stack
- Java
- Spring Boot
- WebSocket / REST
- PostgreSQL
- Redis (if used)
- JWT Authentication

## Features
- User Authentication (JWT)
- Role-based access
- Real-time messaging
- Secure password hashing (BCrypt)

## How to Run
1. Clone the repo
2. Configure application.properties
3. Run DemoApplication
